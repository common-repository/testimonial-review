== Description ==

Testimonial Review plugin is a simple tool to display your customer's testimonials on your WordPress website. You can use any type of feedback to make a testimonial with the author's name, photo, company logo, and a URL to the client's website.

== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as regular WordPress plugin.
2. Go your Plugins setting via WordPress Dashboard and activate it.
3. After activating the plugin you will see "Testimonial Review" menu below the comment's menu at the left side on WordPress dashboard and find "Create Shortcode" page to generate testimonial shortcode.<br /> 
<br />

<strong>Using FTP</strong>
1.Extract testimonial-review.zip to your computer.
2.Upload the testimonial-review directory to your wp-content/plugins directory.
3.After activating the plugin you will see "Testimonial Review" menu on the left side of WordPress dashboard and find "Create Shortcode" page to generate testimonial shortcode.

<br />
<strong>How to use short-code?</strong><br />
When plugin active & all setup done please copy shortcode on the shortcode page & paste it anywhere in your post, page or widgets section to display this testimonial.
